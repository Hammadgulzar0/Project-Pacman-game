# 🎮 Pacman Game (Java)

## 📌 Description

This is a console-based Pacman game developed in Java using Object-Oriented Programming (OOP). The game runs on a 5×5 grid where the player controls Pacman to move in different directions and collect food items. At the same time, ghosts move randomly on the grid, making the game challenging.

## 🧩 Features

* Pacman movement using (^, v, <, >)
* Random movement of ghosts
* Food collection with score system
* Walls that block movement
* Game Over and Win conditions

## 🚀 How to Run

Run `PacmanGame.java` in any Java IDE and play using keyboard inputs.
